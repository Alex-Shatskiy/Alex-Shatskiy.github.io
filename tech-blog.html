What are the differences between relative, absolute, and fixed positioning?

Introduction

There are 3 different types of positioning going to be pointed out in this blog, however in total there are 5 different position styles you can use. The main 3 that are going to be explained are relative, absolute and fixed. All these have different impacts on your website. Good chance you will use at least one if not all while building a website. They are great tools to use to override the basic document flow you’d otherwise get, and give it more of a unique look.

The 5 diffrent positioning elements

      Static
      Relative
      Fixed
      Absalute
      Sticky

What is relative positioning?

Relative will be the first one we’ll talk about. When setting a position on your website to relative, you will notice that it wont do anything without assigning it a position(top, bottom, right, left). For instance if you add a photo with an additional position such as left:300px, the photo will move 300px to the left from its default position, as you will be able to see in the example below.

When the element is moved the position of other elements won’t be affected. It will overlay other elements, as shown below.

What is absolute positioning?

This element type will let you place your element exactly where you want it. Using an absolute element will break the flow of the page and let you position anywhere you want on the document, as you can see the fox is positioned outside the box and ignoring flow of the document.

However, while using absolute if it is in a box that has been set to relative, setting an element to absolute will contain it in such area, as you see below, think of it as the “box” as a parent, and anything inside it is being told what it can and can not do, like the image not being able to leave the balck square.

What is fixed positioning?

Fixed position would be similar to relative position where its taken out of the normal flow of the document however when it comes to resizing the web browser, instead of it resizing relative to the size of the window it will forever be fixed in that position that you have assigned it to.

For instance if you have assigned a photo 200px up and 200px to the left, regardless of what's around it will be at set position, another example would be if the page has a logo at the top and it’s always at the top even if the pages is being scrolled down said logo would be fixed in that position no matter where you are on the page.
